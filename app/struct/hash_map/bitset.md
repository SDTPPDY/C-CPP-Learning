`bitset` 是 C++ 标准库中一个非常实用的**固定大小位集合**容器，位于 `<bitset>` 头文件中。它把每一位（bit）当作一个布尔值使用，内存紧凑、操作高效，特别适合处理**标志位集合**、**有限范围内的整数集合**或**状态压缩**等问题。

---

## 1. 基本概念

`bitset` 的模板参数是编译时常量 `N`，表示它包含 `N` 个二进制位（bit）。每个位只能是 `0` 或 `1`，可以通过下标或成员函数访问和修改。

```cpp
#include <bitset>
std::bitset<8> bs;   // 8 位，所有位初始化为 0
```

---

## 2. 声明与初始化

```cpp
std::bitset<8> b1;               // 全 0
std::bitset<8> b2(0b10101010);   // 用整数初始化，低位对应索引 0
std::bitset<8> b3("10101010");   // 用字符串初始化，索引 0 对应字符串最右字符
std::bitset<8> b4(b2);           // 拷贝构造
```

---

## 3. 常用成员函数

| 函数                             | 说明                                                                       |
| -------------------------------- | -------------------------------------------------------------------------- |
| `set(pos)`                     | 将第 `pos` 位设为 1（默认所有位设为 1）                                  |
| `reset(pos)`                   | 将第 `pos` 位设为 0（默认所有位设为 0）                                  |
| `flip(pos)`                    | 翻转第 `pos` 位（默认所有位翻转）                                        |
| `test(pos)`                    | 返回第 `pos` 位的值（会进行边界检查）                                    |
| `count()`                      | 返回值为 1 的位数                                                          |
| `size()`                       | 返回总位数（即模板参数 `N`）                                             |
| `any()`                        | 是否有任意一位为 1                                                         |
| `none()`                       | 是否所有位都为 0                                                           |
| `all()`                        | 是否所有位都为 1（C++11 起）                                               |
| `to_string()`                  | 返回字符串表示，最左字符对应最高位                                         |
| `to_ulong()` / `to_ullong()` | 转换为 `unsigned long` 或 `unsigned long long`（若超出范围则抛出异常） |

---

## 4. 位操作与运算符

`bitset` 支持位运算符，行为与整数位运算一致：

```cpp
std::bitset<8> a("1100");
std::bitset<8> b("1010");

std::bitset<8> c = a & b;   // 按位与 → 1000
std::bitset<8> d = a | b;   // 按位或 → 1110
std::bitset<8> e = a ^ b;   // 按位异或 → 0110
std::bitset<8> f = ~a;      // 按位取反 → 0011
std::bitset<8> g = a << 2;  // 左移 → 110000
std::bitset<8> h = a >> 1;  // 右移 → 0110
```

也可以通过 `[]` 访问或修改单个位（不做边界检查）：

```cpp
bs[0] = 1;          // 设置最低位
bool bit = bs[3];   // 获取第 3 位
```

---

## 5. 特点与优势

- **内存极致紧凑**：每个位只占 1 比特，`bitset<200001>` 仅占用约 25KB 内存（200001 / 8 ≈ 25000 字节）。
- **操作极快**：位运算和成员函数通常被编译器优化为一条或几条机器指令。
- **编译时固定大小**：大小在编译时确定，无动态内存分配，性能可预测。
- **类型安全**：模板参数确保大小一致才能进行位运算，避免逻辑错误。

## 6. 典型应用场景

### 6.1 有限范围整数集合（如 LeetCode 575 分糖果）

当整数范围已知且不大（如 `[-10⁵, 10⁵]`）时，可用 `bitset` 代替哈希集合：

```cpp
bitset<200001> seen;
for (int x : candyType) {
    seen.set(x + 100000);   // 偏移到非负索引
}
int distinct = seen.count();
```

### 6.2 状态压缩（如 DP 中的子集表示）

用 `bitset` 表示集合，利用位运算快速进行交、并、子集判断等操作：

```cpp
bitset<10> subset;   // 表示一个子集
if ((subset & another) == subset) { /* 子集关系 */ }
```

### 6.3 标志位管理

如记录某个对象的多个布尔属性，比 `vector<bool>` 更高效且易用。

### 6.4 布隆过滤器（简易版）

可以组合多个哈希函数，利用 `bitset` 实现基础布隆过滤器。

## 7. 与 `vector<bool>` 和 `unordered_set` 的比较

| 特性   | `bitset`           | `vector<bool>`                 | `unordered_set`    |
| ------ | -------------------- | -------------------------------- | -------------------- |
| 大小   | 编译时常量           | 运行时可变                       | 运行时可变           |
| 内存   | 1 bit/元素           | 1 bit/元素（但实现可能占用更多） | 每个元素开销大       |
| 操作   | 位运算极快           | 位运算较慢，部分实现有特化       | 哈希操作，O(1) 平均  |
| 适用   | 固定大小，密集位操作 | 需要动态大小                     | 通用集合，不限于整数 |
| 灵活性 | 低（大小固定）       | 中                               | 高                   |

---

## 8. 注意事项

- `N` 必须是编译时常量（如字面量、`constexpr` 变量），不能是运行时变量。
- 使用 `[]` 访问时不检查边界，需确保索引 < N，否则行为未定义。
- 转换 `to_ulong()` 时，若值超出 `unsigned long` 范围会抛出 `std::overflow_error`。
- `bitset` 不支持迭代器，不能用于基于范围的 for 循环。

---

## 9. 示例代码：快速统计不同数字

```cpp
#include <bitset>
#include <vector>

int countDistinct(const std::vector<int>& nums, int minVal, int maxVal) {
    int offset = -minVal;                 // 使最小值映射到索引 0
    std::bitset<200001> bs;               // 假设范围长度 ≤ 200001
    for (int x : nums) {
        bs.set(x + offset);               // 将值映射到位
    }
    return bs.count();                    // 不同值的个数
}
```

---

## 10. 总结

`bitset` 是 C++ 中一种轻量级、高效的位集合工具，特别适合在**范围固定、需要极致内存和速度**的场景下替代 `unordered_set` 或 `vector<bool>`。它的主要优势在于内存占用小、操作速度快、语法简洁，是算法竞赛和底层优化中的常用利器。
